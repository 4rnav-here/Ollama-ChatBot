# Ollama-ChatBot

This is a simple chatbot made using LangChain framework and Llama2 model.


You can run this by simply cloning the entire repositoy and then initiating a virtual enviorment in your system.

Also note that this is a simple chatbot not optimized for performance but made simply for learning purposes so is lacking the speed displayed by the usual models.

- To initiate a virtual enviorment type <mark>env/Scripts/activate<mark>
-  Run <mark>pip install -r requirements.txt</mark> to install the necessary packages.
- To run the code simply type <mark>streamlit run app.py</mark>
